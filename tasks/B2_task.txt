Напиши модуль для Telegram-бота на aiogram 3.x: меню кафе на reply-клавиатуре.

ВАЖНО: только модуль с обработчиками, запуск polling — под if __name__ == "__main__".

Требуются на уровне модуля:

1. router = Router()

2. MENU = {"Кофе": 150, "Чай": 100, "Десерт": 250}  (цены в рублях)

3. def build_menu_keyboard() -> ReplyKeyboardMarkup
   Клавиатура: по одной кнопке на позицию меню (текст кнопки = название),
   плюс отдельной строкой кнопка "Корзина". resize_keyboard=True.

4. async def cmd_start(message) — на /start отвечает
   "Добро пожаловать! Выберите позицию:" с клавиатурой из build_menu_keyboard().

5. async def handle_menu_choice(message) — на текст, совпадающий с позицией
   меню, отвечает: "{название} — {цена} руб. Добавлено в заказ."

6. def format_price(item: str) -> str
   Возвращает строку "{название} — {цена} руб." или "Нет такой позиции",
   если item не из MENU.

Ответы через await message.answer(...).
