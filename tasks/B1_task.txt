Напиши модуль обработчиков для Telegram-бота на aiogram 3.x
(визитка-эхо для клиента).

ВАЖНО: выдай только модуль с обработчиками, пригодный для тестирования.
Запуск polling — строго под if __name__ == "__main__".

Требуются на уровне модуля:

1. router = Router()  (из aiogram)

2. async def cmd_start(message) — обработчик команды /start.
   Отвечает РОВНО текстом: "Привет! Я бот-помощник. Напиши /help для списка команд."

3. async def cmd_help(message) — обработчик команды /help.
   Отвечает РОВНО текстом: "Доступные команды:\n/start — приветствие\n/help — эта справка"

4. async def echo(message) — обработчик любого прочего текстового сообщения.
   Отвечает текстом: "Вы написали: {текст сообщения}".
   Если message.text равен None (стикер, фото) — отвечает:
   "Я понимаю только текст."

Все ответы — через await message.answer(...).
Обработчики зарегистрируй на router декораторами.
