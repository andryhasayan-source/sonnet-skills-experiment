Напиши Python-декоратор retry для повторных попыток при сбоях
(типовая задача для нестабильных внешних API).

Сигнатура:
retry(times: int = 3, delay: float = 0.1, exceptions: tuple = (Exception,))

Поведение:
- Декорированная функция вызывается до times раз.
- Перехватываются только исключения из кортежа exceptions;
  прочие исключения пролетают сразу без повторов.
- Между попытками пауза delay секунд (time.sleep).
- Если все times попыток упали — поднимается последнее исключение.
- При успехе возвращается результат функции, лишних попыток не делается.
- Декоратор должен сохранять имя и docstring исходной функции (functools.wraps).
- times < 1 — поднимать ValueError при создании декоратора.

Выдай один файл кода без примеров использования.
