Напиши Python-модуль работы с интервалами занятости для сервиса
бронирования.

Интервал — кортеж (start: int, end: int) в минутах от начала суток,
start < end.

Требуются функции:

1. merge_intervals(intervals: list[tuple]) -> list[tuple]
   Объединяет пересекающиеся и соприкасающиеся интервалы
   ((60,120) и (120,180) -> (60,180)). Результат отсортирован по start.
   Пустой список -> пустой список.

2. free_slots(busy: list[tuple], day_start: int, day_end: int) -> list[tuple]
   Возвращает свободные интервалы внутри [day_start, day_end)
   с учётом занятых busy (busy может быть неотсортированным
   и с пересечениями). Занятость за пределами дня обрезается.

3. find_slot(busy: list[tuple], duration: int, day_start: int, day_end: int) -> tuple | None
   Возвращает самый ранний свободный интервал длиной ровно duration
   минут, либо None.

Выдай один файл кода без примеров использования.
