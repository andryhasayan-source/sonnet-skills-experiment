Напиши Python-класс RateLimiter для ограничения частоты запросов
методом скользящего окна (типовая задача для ботов и API-клиентов).

Класс:
RateLimiter(max_calls: int, window_seconds: float)

Методы:

1. allow(key: str, now: float = None) -> bool
   Регистрирует попытку для ключа key (например, ID пользователя)
   и возвращает True, если за последние window_seconds по этому ключу
   было меньше max_calls обращений (текущее включается в счёт),
   иначе False — и тогда попытка НЕ учитывается в истории.
   Параметр now — текущее время в секундах (time.time());
   если None — берётся time.time(). Параметр нужен для тестируемости.

2. remaining(key: str, now: float = None) -> int
   Сколько обращений ещё доступно ключу в текущем окне.

3. reset(key: str) -> None
   Сбрасывает историю по ключу.

Требования:
- Разные ключи учитываются независимо.
- Старые записи за пределами окна должны вычищаться из памяти.
- max_calls < 1 или window_seconds <= 0 — ValueError в конструкторе.

Выдай один файл кода без примеров использования.
