Напиши Python-модуль для фильтрации лог-файла веб-сервера.

Формат строки лога:
2026-06-01 12:30:45 | ERROR | payment | Card declined
(дата время | уровень | модуль | сообщение, разделитель " | ")

Требуются функции:

1. parse_line(line: str) -> dict | None
   Разбирает строку в словарь с ключами: timestamp (str), level (str),
   module (str), message (str). Если строка не соответствует формату — None.

2. filter_logs(lines: list[str], level: str = None, module: str = None) -> list[dict]
   Возвращает разобранные записи, отфильтрованные по уровню и/или модулю
   (если параметр None — фильтр по нему не применяется).
   Битые строки пропускаются.

3. count_by_level(lines: list[str]) -> dict
   Словарь {уровень: количество записей}.

Выдай один файл кода без примеров использования.
