Напиши Python-класс TTLCache — кэш с ограничением размера и временем
жизни записей (типовая задача для ускорения API-клиентов).

Класс:
TTLCache(max_size: int, ttl_seconds: float)

Методы:

1. set(key, value, now: float = None) -> None
   Кладёт значение. Если ключей становится больше max_size —
   вытесняется самый давно НЕ ИСПОЛЬЗОВАННЫЙ ключ (LRU-политика;
   использование = set или успешный get).

2. get(key, now: float = None)
   Возвращает значение, если ключ есть и не протух по TTL,
   иначе None. Протухшая запись при обращении удаляется.
   Успешный get обновляет позицию ключа в LRU-порядке.

3. __len__ - количество живых (непротухших) записей на момент вызова.

Параметр now во всех методах — текущее время в секундах;
если None — time.time(). Нужен для тестируемости.
max_size < 1 или ttl_seconds <= 0 — ValueError в конструкторе.

Выдай один файл кода без примеров использования.
