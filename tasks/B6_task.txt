Напиши модуль для Telegram-бота на aiogram 3.x: игра "Угадай число".

ВАЖНО: только модуль, запуск polling — под if __name__ == "__main__".

Требуются на уровне модуля:

1. router = Router()

2. games = {}  — словарь {user_id: загаданное число}

3. def start_game(user_id: int, rng=None) -> int
   Загадывает число 1-10 (random.randint или переданный rng.randint),
   кладёт в games, возвращает его.

4. def check_guess(user_id: int, guess: int) -> str
   Возвращает одно из: "Игра не начата. Напишите /game",
   "Меньше!", "Больше!", "Угадали!".
   Подсказки "Меньше!"/"Больше!" — относительно загаданного
   (guess больше загаданного -> "Меньше!").
   При "Угадали!" игра пользователя удаляется из games.

5. async def cmd_game(message) — на /game вызывает start_game
   и отвечает "Я загадал число от 1 до 10. Угадай!"

6. async def handle_guess(message) — на текстовое сообщение:
   если текст не целое число — отвечает "Отправьте число от 1 до 10.";
   иначе отвечает результатом check_guess.
