Напиши Python-модуль для отчёта по продажам из CSV-файла.

Формат CSV (разделитель — запятая, первая строка — заголовок):
date,city,product,amount
2026-01-15,Москва,Ноутбук,55000
2026-01-16,Казань,Мышь,1200

Требуются функции:

1. load_sales(path: str) -> list[dict]
   Читает CSV и возвращает список словарей с ключами date, city, product, amount.
   amount должен быть приведён к float. Некорректные строки (нет нужных полей,
   amount не число) молча пропускать.

2. total_by_city(sales: list[dict]) -> dict
   Возвращает словарь {город: суммарная выручка}.

3. top_product(sales: list[dict]) -> str | None
   Возвращает название товара с наибольшей суммарной выручкой.
   Если данных нет — None.

Кодировка файла UTF-8. Выдай один файл кода без примеров использования,
точка входа не нужна.
