Напиши Python-модуль валидации конфигурационного файла JSON для клиентского проекта.

Конфиг должен иметь структуру:
{
  "app_name": str (непустая),
  "port": int (1-65535),
  "debug": bool,
  "database": {
      "host": str (непустая),
      "port": int (1-65535)
  },
  "admins": список email-строк (минимум 1 элемент, каждый содержит "@" и ".")
}

Требуются функции:

1. load_config(path: str) -> dict
   Читает JSON-файл. Если файл отсутствует или JSON битый —
   поднимает ValueError с понятным сообщением.

2. validate_config(config: dict) -> list[str]
   Возвращает список текстовых описаний всех найденных ошибок
   (отсутствующие поля, неверные типы, значения вне диапазона).
   Если конфиг валиден — пустой список. Функция не должна падать
   ни на каком входном словаре.

Выдай один файл кода без примеров использования.
