Напиши Python-модуль нормализации российских телефонных номеров
для CRM клиента.

Требуются функции:

1. normalize_phone(raw: str) -> str | None
   Приводит номер к формату +7XXXXXXXXXX (11 цифр после плюса).
   Принимает варианты: 8 905 123-45-67, +7(905)1234567, 79051234567,
   9051234567 (считается мобильным, добавляется +7) и т.п. —
   из строки берутся только цифры.
   Если после очистки получается не российский номер
   (неверная длина, не начинается с 7/8 при 11 цифрах) — None.
   На None и пустой строке не падать, возвращать None.

2. normalize_list(raws: list[str]) -> dict
   Возвращает {"valid": [нормализованные уникальные номера в порядке
   первого появления], "invalid": [исходные строки, которые не удалось
   нормализовать]}.

Выдай один файл кода без примеров использования.
