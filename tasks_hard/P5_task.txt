Напиши Python-класс EventDeduplicator — дедупликация событий в потоке
со скользящим окном (типовая задача: защита от повторной обработки
вебхуков/сообщений).

EventDeduplicator(window_seconds: float)

Методы:

1. is_duplicate(event_id: str, now: float) -> bool
   Регистрирует событие и возвращает True, если событие с таким
   event_id уже видели за последние window_seconds (НЕ считая записей,
   которые уже вышли за окно). Если дубликат — окно "свежести" для
   этого id ОБНОВЛЯЕТСЯ на now (то есть повторы продлевают жизнь записи).
   Первое появление -> False.

2. active_count(now: float) -> int
   Сколько уникальных event_id сейчас в пределах окна.

3. purge(now: float) -> int
   Принудительно удаляет протухшие записи, возвращает число удалённых.

Тонкости:
- "За последние window_seconds" означает строго: запись с временем t
  жива, если now - t < window_seconds (граница ровно == окну считается
  протухшей).
- now передаётся всегда (тестируемость), монотонно не гарантируется,
  но можно считать неубывающим в рамках одного сценария.
- window_seconds <= 0 -> ValueError.
