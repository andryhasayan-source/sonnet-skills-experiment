Напиши Python-функцию pretty_duration для человекочитаемого
форматирования длительности (типовая задача для отчётов/логов).

pretty_duration(seconds: float, max_units: int = 2) -> str

Правила:
1. Единицы: d (дни), h (часы), m (минуты), s (секунды).
2. Показываем не более max_units самых старших НЕНУЛЕВЫХ единиц
   подряд, начиная со старшей ненулевой.
   90061 c = 1d 1h (это 1 день 1 час 1 минута 1 секунда, но max_units=2
   -> "1d 1h", младшие отбрасываются, БЕЗ округления).
3. Если все единицы по нулям (seconds < 1) -> "0s".
4. Дробные секунды отбрасываются (floor) до целого числа секунд
   перед расчётом. Отрицательное число -> ValueError.
5. max_units < 1 -> ValueError.
6. Пропуски нулей ВНУТРИ выбранного окна units не происходят:
   3601 c = "1h 0m" при max_units=2 (час есть, минут ноль, но это
   вторая по старшинству единица после старшей ненулевой — показываем).
   Уточнение алгоритма: найди старшую ненулевую единицу, затем выведи
   подряд max_units единиц начиная с неё (включая нулевые внутри),
   но не выводя единицы младше секунд. Если до конца (s) осталось
   меньше max_units единиц — выводим сколько есть.
   Примеры: 86400 -> "1d 0h"; 3661 -> "1h 1m"; 61 -> "1m 1s";
   59 -> "59s"; 3600 -> "1h 0m".

Выдай один файл кода.
