Напиши Python-вычислитель арифметических выражений (типовая задача —
калькулятор в боте/CLI).

evaluate(expr: str) -> float | None

Поддерживает: целые и десятичные числа (точка), бинарные + - * /,
унарный минус, круглые скобки, любые пробелы. Приоритет стандартный,
скобки важнее, * / левоассоциативны, + - левоассоциативны.

Требования:
1. Деление на ноль -> None (не исключение).
2. Любая синтаксическая ошибка (непарные скобки, два оператора подряд,
   пустые скобки, посторонние символы, пустая строка) -> None.
3. Унарный минус работает в "-5", "3 * -2", "-(2+3)", "2--3" (второй
   минус унарный) -> 5.
4. Результат всегда float. "2+2" -> 4.0.
5. Не использовать eval/exec и любые их аналоги (ast.literal_eval тоже
   нельзя — нужен собственный парсер). Наличие eval в коде = провал.
6. На не-str входе -> None.

Выдай один файл кода.
