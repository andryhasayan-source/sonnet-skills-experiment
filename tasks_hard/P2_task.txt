Напиши Python-функцию глубокого слияния словарей конфигурации
(типовая задача: дефолтный конфиг + пользовательские оверрайды).

deep_merge(base: dict, override: dict) -> dict

Правила:
1. Возвращает НОВЫЙ словарь; base и override не мутируются
   (ни на одном уровне вложенности).
2. Если по одному ключу в обоих словарях лежат dict — сливаются
   рекурсивно. Если хотя бы один не dict — значение из override
   побеждает целиком.
3. Если по ключу в обоих лежат list — НЕ конкатенируются и не
   сливаются поэлементно: список из override заменяет список из base
   целиком (но возвращается его копия, не ссылка).
4. Ключи, которые есть только в base или только в override, сохраняются.
5. Особый случай: если значение в override равно специальному маркеру
   DELETE (определи как DELETE = object() на уровне модуля), то этот
   ключ должен ОТСУТСТВОВАТЬ в результате (удаление ключа), даже если
   он был в base.
6. Глубина вложенности произвольная.

Выдай один файл кода. Экспортируй deep_merge и DELETE.
